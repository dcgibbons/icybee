Welcome to the IcyBee Client Development Page!

Click here to launch IcyBee!

This ICB client uses Sun's Java 2 platform, version 1.5 or higher, with 1.6 as default. All platforms with a Java run-time environment should be able to run this client.

After launching the client for the first time, your Java Run-Time environment will prompt you to create an application shortcut on your desktop. You may either run IcyBee from this shortcut in the future or return to this web page.

Development Resources

IcyBee is an open source project using the GNU Public License.

The project is hosted on SourceForge at https://sourceforge.net/projects/icybee/

Source code may be checked out from Subversion using the following command:
svn co https://icybee.svn.sourceforge.net/svnroot/icybee icybee

Alternatively, the subversion repository may be browsed at http://icybee.svn.sourceforge.net/viewvc/icybee
